Notes 
For any pattern program we have to follow the following three steps
	1 - We need to make use of nested "for loop" concept 
	2 - Decide the number of rows and number of columns
	3 - Decide What is the character that needs to be printed within the Printing
		(S.O.P) Statement
	4 - In the nested for loop the outer for loop will decide the number of rows 
		and the inner for loop will decide the number of columns
